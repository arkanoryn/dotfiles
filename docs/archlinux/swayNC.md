## SwayNC Configuration

SwayNC is a notification center for Wayland compositors. The configuration includes:

- **config.json**: Main configuration file defining the notification center's behavior and appearance

### Key Features

- **Positioning**: Centered at the top of the screen
- **Widgets**: Includes DND (Do Not Disturb), buttons grid, MPRIS (media player), volume, backlight, and notifications
- **Customization**: Various settings for appearance and behavior

## Integration with Other Components

SwayNC works seamlessly with other components in this setup:

- **Hyprland**: SwayNC integrates with Hyprland for notification management. See [Hyprland documentation](hyprland.md) for details.
- **Waybar**: SwayNC complements Waybar for system status monitoring. See [Waybar documentation](waybar.md) for configuration details.
- **Common Tools**: SwayNC works with common tools like Fish shell and Tmux. See the [Common Configuration documentation](../common/README.md) for more information.

## Troubleshooting

If you encounter issues with SwayNC:

1. **Check Logs**: Review SwayNC logs for errors
2. **Verify Dependencies**: Ensure all required packages are installed
3. **Review Configuration**: Check for syntax errors in configuration files
4. **Consult Documentation**: Refer to the official SwayNC documentation
